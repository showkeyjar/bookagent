# 📚 BookAgent 快速开始

## 🚀 30秒启动

### 第一步：下载项目
```bash
git clone https://github.com/yourusername/bookagent.git
cd bookagent
```

### 第二步：智能启动
```bash
# Windows 用户
start.bat

# Mac/Linux 用户
./start.sh
```

**就这么简单！** 🎉

系统会自动：
- 🔍 检测你的环境
- 📦 安装必要依赖
- 🚀 选择最佳启动模式
- 🌐 打开浏览器

## 🛠 如果遇到问题

### 快速诊断
```bash
# 系统检查
python check_system.py

# 功能测试
python test_system.py
```

### 备用启动方式
```bash
# 纯后端模式（最稳定）
python start_backend_only.py

# 智能启动
python smart_start.py
```

## ✨ 首次使用指南

### 1. 打开浏览器
启动成功后会自动打开 http://localhost:3000

### 2. 创建第一本书
- 点击 "开始我的第一本书"
- 输入书名和简介
- 选择合适的模板
- 开始创作！

### 3. 与AI助手协作
- 在编辑器右侧找到AI助手
- 告诉它你想写什么
- 让它帮你生成内容、优化表达

## 🎯 核心功能

### 📝 智能写作
- **思路整理**：AI帮你理清写作思路
- **内容生成**：基于大纲自动生成章节内容  
- **表达优化**：让你的文字更加精准专业

### 📊 进度跟踪
- **实时统计**：字数、章节数、完成度
- **写作习惯**：每日写作量、连续天数
- **智能提醒**：个性化的写作建议

### 🎨 专业输出
- **多种格式**：PDF、Word、HTML
- **精美排版**：专业的图书格式
- **一键导出**：无需复杂操作

## 💡 使用技巧

### 与AI对话的技巧
- **具体描述**：告诉AI具体的写作需求
- **提供背景**：让AI了解你的读者群体
- **分步骤问**：复杂问题分解为小问题

### 高效写作建议
- **先大纲后内容**：用AI帮你完善章节结构
- **专注一章**：一次只专注一个章节
- **定期保存**：系统会自动保存，但建议手动保存

## 🆘 遇到问题？

### 常见问题
- **启动失败**：检查Python是否安装（需要3.8+）
- **页面打不开**：等待30秒让服务完全启动
- **AI不响应**：刷新页面重试
